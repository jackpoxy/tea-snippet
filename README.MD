## 简介  

tea 组件的代码片段，支持绝大部份的组件

![intro](https://raw.githubusercontent.com/jackpoxy/tea-snippet/master/images/intro.gif)

## 使用方法  

可以使用「tea-组件名」的方式来调用，例如：`tea-button`
